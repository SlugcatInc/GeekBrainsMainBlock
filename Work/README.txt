1. Создаем массив строк inputArray и заполняем его значениями "Hello", "2", "world"
2. Вызываем метод FilterArray и передаем ему inputArray
3. В методе FilterArray считаем количество элементов в inputArray, у которых длина строки меньше или равна 3
4. Создаем новый массив resultArray, размер которого равен count (количеству элементов с длиной строки <= 3)
5. Заполняем resultArray элементами из inputArray, удовлетворяющими условию (длина строки <= 3)
6. Выводим на экран "Filtered array:" и элементы resultArray, каждый с новой строки
7. Программа завершает свою работу
